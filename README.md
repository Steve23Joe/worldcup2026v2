# WC2026 Dashboard

Static World Cup prediction dashboard powered by local data pipeline.

## Architecture

```
Python pipeline (agent repo) → export_web_data.py → public/data/*.json
                                                         ↓
                                               Vue 3 + Vite static site
                                               (no backend, no API calls)
```

The frontend reads ONLY static JSON files from `public/data/`. No external
API calls. No backend. No API keys.

## Quick Start

```bash
npm install
npm run dev        # http://localhost:5173
npm run build      # outputs to dist/
```

## Data Refresh

Data files are generated by the Python pipeline in the agent repository
and copied here via the publish workflow:

```bash
python3 scripts/run_publish_dashboard.py
```

## Deployment (EdgeOne)

```bash
npm install && npm run build   # Build command
dist                           # Output directory
```

## Security

- **No API keys**: Zero keys, tokens, or secrets in the frontend.
- **Static JSON only**: All data from `public/data/*.json`.
- **No external calls**: No fetch to external domains.
- **No .env**: No environment variables.

## Disclaimer

**Analytical output only. Not betting advice.**
